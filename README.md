# OpenClaw Mission Control

Professional local dashboard for managing OpenClaw agents, API budgets, and skills with a clean dark UI.

## Features

- Mission dashboard with live-style agent cards and progress chart
- API key management and credit monitoring
- Activity timeline with event logging
- Agent builder with model/tool configuration
- Skill marketplace and installed skills control panel
- Local persistence via browser `localStorage`

## Run in VS Code

1. Open this folder in VS Code.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open `http://localhost:5173`.

## Build

- `npm run build`
- `npm run preview`

## Notes

- No backend is required.
- This is built for local personal usage and can be extended with direct OpenClaw CLI and gateway integration.
